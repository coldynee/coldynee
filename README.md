## 👨‍💻 О себе

Начинающий iOS-разработчик с полным циклом создания приложения в портфолио. Разрабатываю и поддерживаю проект **ResumeHub** — платформу для публикации резюме и поиска работы.

Самостоятельно проектирую архитектуру (**MVVM+Coordinator**), интегрирую **Firebase**, реализую аутентификацию и отправку email-кодов через собственный **SMTP-клиент** (без сторонних SDK).

🎯 **Цель:** найти Junior-позицию, где смогу решать реальные задачи, быстро расти и приносить пользу. <a href="https://hh.ru/resume/98a60735ff1063afb40039ed1f4c62676e7945?from=share_ios">Резюме</a>

## 📱 Мой главный проект

### [ResumeHub](https://github.com/coldynee/ResumeHub)

Платформа для публикации резюме и поиска работы.

**Ключевые возможности:**
- ✅ Регистрация и вход (логин/пароль + одноразовый код на email)
- ✅ Восстановление пароля с генерацией нового
- ✅ Firebase Firestore для хранения пользователей
- ✅ Собственный SMTP-клиент (без сторонних SDK)
- ✅ Анимированный Launch Screen
- ✅ Локализация (русский / английский)
- ✅ Светлая и тёмная темы

**Архитектура:** MVVM + Coordinator, Combine, UIKit (SnapKit)
